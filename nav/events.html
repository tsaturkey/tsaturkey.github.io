Events

Fashion Design

Participants research, design, create a portfolio, and a wearable prototype relating to the current year’s theme. Eda Kahraman, Berra Öztan, Ceren Abanoz, and Lara Turgut are designing garments of 80’s style and fashion trends for this year’s conference.

Biotechnology Design

In Biotechnology Design the aim is to present biotechnological concepts and methods related to the year’s theme, this year’s theme being ‘food’. This year we’re participating as a team of four with the participants Zeynep Duygulu, Deniz Ece Sönmez, Emre Güler and Lara Turgut.

System Control Technology

System Control Technologies is an event, in which one should be able to think practically. But since this is a group event, one must also know how to communicate and develop ideas in a group. Because the goal of the event is to build a robot, which is able to solve the given problem. To achieve this purpose, a group has to fulfill two subtasks: building the robot and coding the robot. If these two sub, each working a subtask, cannot communicate properly, the end product will not be able fulfill the task, no matter how well the particular subtask are done. So this event also measures the group work skills besides technical and practical skills.

Webmaster

The challenge of the Webmaster event is to design and build a website based on the year’s theme, this year’s theme being ‘artificial intelligence’. The participants of this year are Lara Turgut, Zeynep Duygulu, and Ömer Yağmurlu.
